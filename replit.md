# BrewDash Website

## Overview
A clean, modern, premium website for BrewDash - a campus-first tea brand serving authentic tribal Ada tea and traditional flavors to university students across India.

## Project Details
- **Brand**: BrewDash
- **Tagline**: Great Tea for Great Minds
- **Target Audience**: Students, Professors, Startup Founders, and Tea Lovers
- **Location**: Near DU Gate, Delhi

## Technology Stack
- **Frontend**: Single HTML file with embedded CSS
- **Styling**: CSS Grid, Flexbox, Google Fonts (Poppins)
- **Server**: Python HTTP Server on port 5000
- **No JavaScript**: Pure HTML + CSS implementation

## Design System
### Brand Colors
- **Navy Blue**: `#153A68` (primary color)
- **Teal Green**: `#29B7A6` (accent color)
- **White**: `#ffffff` (background)
- **Cream**: `#fdfbf7` (alternate background)

### Design Features
- Clean sans-serif typography (Poppins)
- Rounded corners on all cards and buttons
- Soft shadows for depth
- Smooth hover animations (0.3s transitions)
- Fully responsive for mobile, tablet, and desktop

## Website Sections

1. **Sticky Header** - Navigation with logo and menu links
2. **Hero Section** - Main tagline with CTA buttons
3. **Value Proposition** - Brand positioning statement
4. **Vision & Mission** - Two-column grid with teal-bordered cards
5. **About BrewDash** - Brand introduction
6. **Tea Menu** - 4-card grid showcasing products (₹10-₹35)
7. **Special Offers** - Rush Hour, Educators', and Founders' deals
8. **Subscriptions** - Weekly and Monthly tea pass options
9. **Why BrewDash** - 6 unique selling points
10. **Brand Story** - Tribal founder's journey
11. **Target Audience** - Icons for different customer segments
12. **CTA Strip** - Full-width navy call-to-action
13. **Footer** - Location, timing, and contact information
